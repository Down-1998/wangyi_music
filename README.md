# 网易云音乐微信小程序

## 启动:音乐serve

  **npm start**
  

## 微信开发这工具中打开MusicWeApp,
 **使用pubsub-js实现页面之间的通信; 使用jwt实现数据加密**

**主要实现首页    视频    个人中心  登陆   每日推荐   歌曲详情   歌曲播放页面等柒个页面,使用的是网易云的真实接口,包括登陆接口**
